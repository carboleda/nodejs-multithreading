# nodejs-multithreading
This an example of using multithread in nodejs

# Install dependencies
```bash
yarn install
```

## How to execute?
```bash
yarn start:single OR start:multi
```

Execution times to build 1.000.000 of user names:
|Mode           | Time in seconds   |
| ------------- | -----------------:|
|Single thread  | 756s              |
|Multi thread   | 155s              |
